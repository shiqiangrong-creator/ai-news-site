# AI 资讯聚合网站 - 部署指南

## 📋 部署前准备

### 1. 准备工具和账号

- GitHub 账号
- Vercel 账号 (推荐使用 GitHub 账号登录)
- DeepSeek API Key

### 2. 获取 DeepSeek API Key

1. 访问 [DeepSeek 开放平台](https://platform.deepseek.com/)
2. 完成注册/登录
3. 进入「API Keys」页面
4. 点击「创建 API Key」
5. 复制并妥善保存生成的 Key

## 🚀 部署步骤

### 方式一：完整部署（推荐）

#### 第一步：Fork 项目

1. 点击本项目右上角的「Fork」按钮
2. 选择你的 GitHub 账号
3. 等待仓库创建完成

#### 第二步：配置 Vercel

1. 登录 [Vercel](https://vercel.com)
2. 点击「Add New」→「Project」
3. 选择刚刚 Fork 的仓库
4. 配置项目设置：
   - **Framework Preset**: Next.js
   - **Root Directory**: `./ai-news-site`
   - **Build Command**: `npm run build`
   - **Output Directory**: `.next`

5. 点击「Environment Variables」添加：
   - Name: `DEEPSEEK_API_KEY`
   - Value: 你的 DeepSeek API Key

6. 点击「Deploy」开始部署

#### 第三步：配置 GitHub Secrets

1. 进入你的 Fork 仓库
2. 点击「Settings」→「Secrets and variables」→「Actions」
3. 点击「New repository secret」，添加以下 Secrets：

| Secret Name | Value | 说明 |
|------------|-------|------|
| DEEPSEEK_API_KEY | sk-xxx... | DeepSeek API Key |
| VERCEL_TOKEN | xxxx | Vercel 访问令牌 |
| VERCEL_ORG_ID | xxxx | Vercel 组织 ID |
| VERCEL_PROJECT_ID | xxxx | Vercel 项目 ID |

#### 第四步：获取 Vercel 配置信息

1. **VERCEL_TOKEN**:
   - 进入 Vercel → Settings → Tokens
   - 点击「Create Token」
   - 选择过期时间和权限范围
   - 复制生成的 Token

2. **VERCEL_ORG_ID 和 VERCEL_PROJECT_ID**:
   - 在项目中运行: `npm i -g vercel`
   - 运行: `vercel link`
   - 或查看 `.vercel/project.json` 文件

### 方式二：本地开发后部署

```bash
# 1. 克隆仓库
git clone https://github.com/你的用户名/ai-news-site.git
cd ai-news-site

# 2. 安装依赖
cd ai-news-site
npm install

# 3. 配置环境变量
export DEEPSEEK_API_KEY="你的API Key"

# 4. 本地运行
npm run dev

# 5. 构建
npm run build

# 6. 部署到 Vercel
npm i -g vercel
vercel --prod
```

## ⚙️ GitHub Actions 配置说明

工作流文件位置：`.github/workflows/update-news.yml`

### 自动执行时间

```yaml
schedule:
  - cron: '0 */6 * * *'  # 每6小时执行一次
```

可选配置：
- 每小时: `cron: '0 * * * *'`
- 每天凌晨: `cron: '0 0 * * *'`
- 每周一: `cron: '0 0 * * 1'`

### 手动触发

在 GitHub 仓库的 Actions 页面，可以手动点击「Run workflow」立即执行。

## 🐛 常见问题

### Q: RSS 抓取失败

**A:** 可能的原因：
1. RSS 源暂时不可用
2. 网络连接问题
3. RSS 格式变化

检查日志输出，手动访问 RSS 地址确认。

### Q: AI 摘要生成失败

**A:** 检查：
1. DEEPSEEK_API_KEY 是否正确设置
2. API 额度是否充足
3. 网络连接是否正常

### Q: Vercel 部署失败

**A:** 常见解决方案：
1. 检查 Build Command 是否正确
2. 确认 Node.js 版本（推荐 18+）
3. 查看 Vercel 部署日志

### Q: 如何查看网站访问量

**A:** 在 Vercel Dashboard 的 Analytics 页面可以查看。
